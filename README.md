# Проект: Место

### Обзор

Учебный проект Место.
В проекте использованы технологии: HTML, CSS, css-grid, css-flexbox, JavaScript

Проект адаптирован под основные размеры экранов устройств: 320p, 768p, 1024p и 1280p
